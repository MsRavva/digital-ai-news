# Project Brief

## Проект

Digital AI News - веб-приложение для публикации и чтения новостей об AI с авторизацией, ролями пользователей, профилями и административными сценариями.

## Цели

- Обеспечить удобную публикацию и просмотр материалов.
- Поддерживать роли `student`, `teacher`, `admin`.
- Сохранить стабильный auth flow, включая email/password и OAuth.
- Упростить поддержку кода за счет централизованных серверных сценариев.

## Границы

- Веб-приложение на Next.js App Router.
- Backend и auth через Supabase.
- UI строится на Tailwind CSS и shadcn/ui.

## Текущий приоритет

- Стабильность post-auth redirect flow.
- Актуализация операционной документации проекта.
