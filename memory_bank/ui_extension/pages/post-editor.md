# Post Editor

## Страницы

- `/create` - создание новой публикации через общую форму `CreatePostForm`.
- `/edit/[id]` - редактирование существующей публикации через `EditPostForm`.

## Основные UI-элементы

- `components/ui/markdown-editor.tsx` - общий редактор Markdown для создания и редактирования постов.
- Markdown toolbar: быстрые действия для жирного текста, курсива, заголовков, цитат, списков, кода, ссылок и изображений.
- Переключение режимов `Редактор / Предпросмотр / Разделить`.

## Вставка изображений

- Изображение выбирается через локальный file picker прямо из тулбара.
- Файл загружается в Supabase Storage bucket `post-images` через защищенный endpoint `/api/uploads/post-image`.
- В Markdown вставляется публичный URL изображения как стандартная конструкция `![alt](src)`.
- Политики Storage ограничивают загрузку, обновление и удаление путей только в собственной папке пользователя.
- Для снижения веса публикаций действует клиентское и серверное ограничение на размер файла до `4 МБ`.
