# Post Editor

## Страницы

- `/create` - создание новой публикации через общую форму `CreatePostForm`.
- `/edit/[id]` - редактирование существующей публикации через `EditPostForm`.

## Основные UI-элементы

- `components/ui/markdown-editor.tsx` - общий редактор Markdown для создания и редактирования постов.
- `components/ui/markdown-content.tsx` - общий рендер Markdown для предпросмотра и страниц публикаций.
- `app/posts/[id]/layout.tsx` - динамическая metadata/OpenGraph-разметка для link preview конкретной публикации.
- Markdown toolbar: быстрые действия для жирного текста, курсива, заголовков, цитат, списков, кода и ссылок.
- Переключение режимов `Редактор / Предпросмотр / Разделить`.
- Нумерованные списки в предпросмотре используют внешние маркеры и сохраняют текст первого абзаца на одной строке с номером.
- Markdown-сепаратор `---` отображается с вертикальным отступом сверху и снизу.

## Вставка изображений

- Автоматическая загрузка изображений из редактора отключена.
- Для изображений используется стандартный Markdown `![alt](src)` с уже доступным внешним URL.
