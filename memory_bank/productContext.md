# Product Context
Платформа для обмена AI-новостями и учебными материалами с ролевой моделью.
- Сценарии: Вход (email/OAuth), чтение/CRUD постов с markdown, админка.
- Проблема: Текст в блоках кода отображался разными шрифтами для кириллицы/латиницы.
- Решение: Интегрирован JetBrains Mono с поддержкой кириллицы для pre/code.
