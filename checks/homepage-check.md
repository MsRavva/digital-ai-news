# Проверка главной страницы на соответствие стилевому гайду

## Элементы для проверки:

### 1. Кнопки
- [ ] Default кнопка (фиолетовая с белым текстом) - используется в форме входа/регистрации
  - Вход: variant="saas"
  - Регистрация: variant="saas-secondary"
- [ ] Secondary кнопка - используется в UserNav для переключения темы
- [ ] Ghost кнопка - используется в UserNav для открытия меню пользователя
- [ ] Icon кнопка - используется во ViewToggle

### 2. Навигация
- [ ] MainNav - основная навигация
- [ ] UserNav - пользовательская навигация

### 3. Компоненты
- [ ] PublicationCategoryTabs - вкладки категорий
- [ ] ViewToggle - переключатель режима просмотра
- [ ] InfinitePostsList - список постов в режиме сетки
- [ ] PaginatedPostsTable - таблица постов в режиме таблицы
- [ ] PostsList - список постов
- [ ] PostsTable - таблица постов

### 4. UI элементы
- [ ] Input - поле поиска
- [ ] Card - карточка для отображения контента
- [ ] SimpleAvatar - аватар пользователя

### 5. Иконки
- [ ] Plus - иконка создания поста
- [ ] Search - иконка поиска
- [ ] X - иконка очистки поиска
- [ ] Grid2X2 - иконка сетки
- [ ] List - иконка списка
- [ ] Sun - иконка светлой темы
- [ ] Moon - иконка темной темы
- [ ] User - иконка профиля
- [ ] FileText - иконка моих публикаций
- [ ] LogOut - иконка выхода

### 6. Специальные элементы
- [ ] saas-window - окно приложения
- [ ] saas-window-header - заголовок окна
- [ ] saas-window-dot - точки управления окном

## Цвета для проверки:
- [ ] --saas-purple - основной фиолетовый цвет
- [ ] --saas-purple-dark - темный фиолетовый цвет
- [ ] --saas-purple-light - светлый фиолетовый цвет

## Темы:
- [ ] Светлая тема
- [ ] Темная тема

## Примечания:
На главной странице нет бейджей, так что их проверять не нужно.
